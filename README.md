# VSLInterpreter
## 运行方式

1. 运行时需要打开RTTI，步骤如下：
   项目上右键->属性->C/C++->语言->启用运行时类型信息
2. 运行后在控制台中输入VSL语句，输入^Z完成输入


## 目录结构
* 源代码均在Chapter2文件夹下。
* 实验过程中完成的设计文档（读书笔记）位于对应的文件目录下
* 测试使用的示例代码以及得到的结果截图位于对应文件夹下
* output.txt为语法分析的输出结果
